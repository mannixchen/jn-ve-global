<h3>优化</h3>

6. 适配 2.9.0 组件库 [详见更新日志](../../components/UpdateLog.md)

1. 应用功能>工作台菜单可配置额外参数及home透传获取
2. Home 增加分类筛选（监管已最新，无需更新，其他组需要更新）
3. uums 增加分类字典管理模块

<h3>更新范围</h3>

1. 基座包
2. 网关、nginx 配置（必须）
3. uums 资源包
4. 所有微应用下的
    - `index.html`
    - `.env`
5. 微应用依赖包升级
    - `"@jsjn/types": "^1.4.3"`
    - `"@jsjn/micro-core-components": "^2.3.1"`
6. 除监管外，需要更新 home 应用的源
